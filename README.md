# Portfolio: Веб-разработчик / SEO-специалист / Контекстная реклама

Это репозиторий моего персонального сайта-портфолио, созданного для демонстрации услуг по веб-разработке, SEO-продвижению и настройке контекстной рекламы.

🔗 **Живая версия сайта:** [aquilongroupltd-stack.github.io/my-portfolio](https://aquilongroupltd-stack.github.io/portfolio/)

## 🛠 Используемые технологии
- HTML5 / CSS3
- JavaScript
- WordPress (разработка тем)
- Адаптивная верстка (mobile first)
- CSS-переменные для кастомизации

## 📌 О проекте
Сайт представляет мои компетенции как технического специалиста, работающего на результат. Здесь я рассказываю о своих услугах, показываю примеры работ и даю возможность связаться для сотрудничества.

### Основные разделы
- **Услуги:** Веб-разработка, SEO, контекстная реклама (PPC)
- **Портфолио:** Примеры реализованных проектов
- **Преимущества:** Почему клиенты выбирают работу со мной
- **Цены:** Прозрачные тарифы на ключевые услуги
- **Контакты:** Telegram для быстрой связи
